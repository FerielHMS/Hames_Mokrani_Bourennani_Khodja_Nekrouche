DESCRIPTION DU PROJET
================================================================================
Tokyo Drift est un site de vente de voitures où les utilisateurs peuvent :
- Parcourir les voitures (Sport, SUV, SEDAN, Luxe, Électrique)
- Acheter des voitures
- Réserver des essais routiers (50$ de dépôt)
- Créer un compte et se connecter

Il y a 3 types d'utilisateurs :
- Client : peut acheter des voitures et réserver des essais
- Travailleur : peut gérer les réservations/commandes + reçoit 10% de réduction
- Admin : gère tout (voitures, travailleurs, utilisateurs, commandes)
